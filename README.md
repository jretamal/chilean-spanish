## English

### Overview
Based on the **chilean-spanish** skill, this project is designed to:
- **Rewrite text** into "Chileno" (Chilean slang).
- **Explain idioms** and local vocabulary.
- **Interpret** colloquialisms for NLP or educational purposes.

### Key Features
- **Tone Handling**: Ranges from formal standard Spanish to highly informal slang ("coa").
- **Conjugation Rules**: Addresses specific verb changes, such as using "cachai" instead of "cachas" or the pronoun "voh".
- **Vocabulary**: Documents common terms like *Weón*, *Al tiro*, *Cachar*, *Bacán*, *Fome*, *Pega*, and *Pololo/a*.

### Installation

```bash
npx skills add https://github.com/jretamal/chilean-spanish --skill chilean-spanish
```

### Usage
Include the skill in your project to process Chilean Spanish text. Refer to the skill documentation for specific implementation details.

## Español

### Descripción General
Basado en la habilidad **chilean-spanish**, este proyecto está diseñado para:
- **Reescribir texto** en "chileno" (jerga chilena).
- **Explicar modismos** y vocabulario local.
- **Interpretar** coloquialismos para propósitos educativos o de procesamiento de lenguaje natural.

### Características Principales
- **Manejo de Tono**: Abarca desde español formal estándar hasta jerga muy informal ("coa").
- **Reglas de Conjugación**: Aborda cambios verbales específicos, como usar "cachai" en lugar de "cachas" o el pronombre "voh".
- **Vocabulario**: Documenta términos comunes como *Weón*, *Al tiro*, *Cachar*, *Bacán*, *Fome*, *Pega* y *Pololo/a*.

### Instalación

```bash
npx skills add https://github.com/jretamal/chilean-spanish --skill chilean-spanish
```

### Uso
Incluye la habilidad en tu proyecto para procesar texto en español chileno. Consulta la documentación de la habilidad para detalles específicos de implementación.
